App launching — call open_app directly:
- "Open X" / "Launch X" → open_app. Always use the official English app name ("Google Chrome", "Finder").
- For category-based requests (user may have set defaults):
  - "Open email" / "email app" → app_name="Mail"
  - "Open browser" → app_name="Safari"
  - "Open terminal" → app_name="Terminal"
  - "Open editor" → app_name="Visual Studio Code"
